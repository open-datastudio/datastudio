Superset¶


Apache Superset is a modern, enterprise-ready business intelligence web application. Open data studio makes it easy to deploy on the cloud.

Key features

  • Click to deploy. No setup required.

  • Postgres and Redis are pre-configured.

  • Connect database on private network using secure tunnel
