Metabase¶

Metabase is the easy, open source way for everyone in the users’ company to ask questions and learn from data. Open data studio makes it easy to deploy on the cloud.

Key features

  • Click to deploy. No setup required.

  • Postgres is pre-configured

  • Connect database on private network using secure tunnel
