Hive metastore¶

Apache Hive Metastore is the central repository of Apache Hive, Apache Spark metadata and more. Open data studio makes it easy to deploy on the cloud.

Key features

  • Based on Hive 3.1.2

  • Click to deploy. No setup required

  • Configured with Postgresql DB

  • Connect from Spark thriftserver and Apache Zeppelin. No configuration required
